## Setup and Usage

xmysql requires node >= 7.6.0

```
npm install -g xmysql
```
```
xmysql -h localhost -u mysqlUsername -p mysqlPassword -d databaseName
```
```
http://localhost:3000
```
example usage by batinay:

```
xmysql -h localhost -u root -p 12345678 -d CommunityPlatform
```
